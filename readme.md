## Журнал изменений

Для просмотра истории используется команда *git log*. Для этого в терминале с папкой-репозиторием нужно написать *git log*. В открывшемся журнале можно найти:
* Хеш (номер) коммита,
* Дату и время коммита,
* Автора коммита,
* Сообщение коммита.
